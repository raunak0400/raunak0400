How It Works

Powered by modern tech.

A lean serverless stack that keeps response times under 100ms globally.

Python

Core backend logic handles auth token management and API requests.

Flask

Lightweight web framework exposes the single /api/orchestrator endpoint.

Spotify API

Fetches your real-time Now Playing or last-played track with OAuth 2.0.

Vercel

Serverless deployment with edge caching for sub-100ms global response times.
